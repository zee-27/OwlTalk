# OwlTalk
OwlTalk is a PHP/MySQL forum-style web application hosted on the Rowan University Elvis server. It allows students to create threads, comment, vote, browse major-specific discussions, manage profiles, upload profile pictures, and send private messages.

Live Server Environment
This project is hosted on the Rowan Elvis web server.

Technologies used
●	PHP
●	MySQL
●	Apache
●	HTML/CSS
●	Rowan Elvis Server

Features
●	User registration and login
●	User profile pages
●	Account dashboard
●	Profile picture uploads
●	Create/Edit/Delete threads
●	Create/Edit/Delete comments
●	Upvote/Downvote posts
●	Upvote/Downvote comments
●	Search threads
●	Category filtering
●	Major-specific discussion board
●	Private messaging system

Server Deployment
Project files were uploaded to the Rowan Elvis server using SecureCRT and SecureFX.
